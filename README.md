# Blockchain-Based-Barangay-Document-Management-System-Public-Documentation-
This repository documents a **secure document management platform** designed for barangay-level record handling.   The system integrates **Blockchain**, **AES encryption**, and **OCR** for secure document verification and traceability.

##  Overview
The project ensures authenticity and immutability of barangay-issued documents by recording and encrypting all transactions and verifications in a distributed ledger.

---

##  Features
- Blockchain transaction recording for audit trail  
- AES encryption for document data protection  
- OCR-based document verification for automated validation  
- Admin and citizen-facing interfaces
